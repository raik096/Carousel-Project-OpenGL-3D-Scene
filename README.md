# 🎢 Carousel Project — OpenGL 3D Scene

## 🖼️ General Description

This project was born out of a personal curiosity for computer graphics. The goal was to develop an interactive C++ application (Linux environment, utilizing OpenGL graphics libraries) by following the steps and tutorials on [The Cherno's YouTube channel](https://www.youtube.com/@TheCherno). The result is an application that displays and allows manipulation of an animated scene containing a car carousel on a realistic terrain, with various graphical elements and lighting.

Starting from a basic OpenGL setup, various features and personal improvements have been implemented to bring the scene to life.

The work was carried out in collaboration with Davide Gaudio [github.com/OneAbowAll] 🖖🏼

---

## 🔧 Implemented Features

### Scene Visualization

- Viewpoint navigation: it is possible to observe the scene from any angle.
- The terrain is generated from a heightmap and displayed with tileable textures.

### Texturing

- The terrain has a tileable texture obtained from `terrain_256.png`, automatically mapped onto the vertices.
- The track is also textured: coordinates are calculated based on the progressive distance from the starting point.

### Meshes and Objects

- Added and correctly rendered:
  - Streetlights
  - Trees
  - Animated cars
  - Cameramen

### Lighting

- **Directional sunlight** integrated into the scene.
- Realistic and coherent **local lighting from streetlights**.
- **Car headlight projection** implemented via projective texturing.

### Shadows

- Shadows generated by all light sources: sun, streetlights, and car headlights.

---

## Additional Points

- Work carried out in a modular way: each graphical or lighting element was tested separately before integration.
- Particular attention was paid to the balance between visual quality and performance.
- Navigable scenes without lag or glitches, with good texture and memory management.

---

## 🔍 Approach Taken

- After an initial phase of setting up the core architecture following the video tutorials, the focus shifted to terrain generation and texture management.
- Later, work continued on adding 3D objects (models already included or found from free sources), setting up materials and lights.
- Finally, the multiple shading and lighting system was integrated, utilizing the concepts learned from the channel and expanding upon them where necessary.
